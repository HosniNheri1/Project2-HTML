# GlobeTravel — Agence de voyage (site vitrine)

Ce projet est une **page web professionnelle** pour une agence de voyage fictive nommée **GlobeTravel**. Il présente une interface moderne, responsive et entièrement statique (HTML / CSS / JS) destinée à démontrer des compétences en **intégration front-end**.

## ✨ Fonctionnalités

- **Présentation des services** : circuits sur mesure, réservation vols & hôtels, assistance 24/7.
- **Mise en avant des destinations** : Paris, Maldives, Tokyo (avec images Unsplash).
- **Offres recommandées** : escapade romantique, aventure, services sur mesure — avec prix et boutons d’appel à l’action.
- **Recherche simplifiée** : champ de recherche (destination + date).
- **Témoignages clients** sous forme de blocs.
- **Formulaire de contact** fonctionnel (interface, sans backend).
- **Menu responsive** : un bouton “☰” permet d’afficher/masquer la navigation sur mobile.
- **Pied de page dynamique** : l’année est mise à jour automatiquement via JavaScript.

## 🛠️ Stack utilisée

- HTML5 sémantique
- CSS3 (via le fichier `fich2.css` – à fournir séparément)
- JavaScript (événements, ouverture/fermeture du menu, date courante)
- Google Fonts (Inter)
- Images libres de droit (Unsplash)

## 🧩 Structure du code

- Un fichier unique `index.html` pour toute la structure.
- Le style est référencé dans `fich2.css` (à créer pour que la mise en page s’affiche correctement).
- Un script interne gère la navigation responsive et l’affichage dynamique de l’année.

## 🚀 Objectif

Ce site illustre ma capacité à construire une **interface utilisateur professionnelle**, responsive, intégrant des bonnes pratiques d’accessibilité (balises `aria-label`, `alt` sur images, structure logique). Il peut servir de base pour un portfolio ou être adapté pour un projet réel.

> ⚠️ Les formulaires de recherche et de contact sont purement **front-end** – pour qu’ils envoient des données, une connexion à un backend ou à un service externe (ex. Formspree) est nécessaire.
